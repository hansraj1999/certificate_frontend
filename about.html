About This Website / WebApp

This project is for the fynd Academy.Most of the certificates which students gets are not verifiable so for a company/college to verify that certification is a hassle. Therefore my aim was to fix this real life issue by making a module that generates certificates in bulk and that certificates are verifiable by either Unique id or by qr code.

Tech-Stack Used to make this WebApp

FastAPI to handle Backend of this WebApp

HTML, CSS, JS, Bootstrap 5: For the Frontend

SQLite : For Database Handling
